ufmcontrol

The ufmcontrol utility is a program to control a radio attached to USB in FreeBSD. It has the same interface of cdcontrol and i18n/l10n support.

The program was tested with D-Link DRU-R100 USB Radio (DSB-R100).

Screenshot

Installation

pkg install ufmcontrol-i18n

or

(cd /usr/ports/audio/ufmcontrol-i18n && make install clean)
